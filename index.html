什么是可信数据源 MCP？

可信数据源 MCP 是一个开源的数据源检索服务，通过 Model Context Protocol 为 AI 提供权威、可信的数据源查找能力。帮助用户快速找到经过验证的、高质量的数据源，确保 AI 使用准确可靠的信息。

🔍 智能检索匹配

基于语义理解的数据源搜索，快速找到最匹配需求的权威数据源

✓ 权威认证保障

精选来自政府、学术机构、行业组织等权威渠道的高质量数据源

📚 开源社区驱动

完全开源透明，社区共建共享，持续更新维护数据源库

⚡ 一键快速接入

标准化 MCP 接口，简单配置即可将数据源接入到 Claude 等 AI 应用

🏷️ 分类标签体系

按领域、类型、地域等维度分类，支持标签过滤和精准查找

📊 质量评分机制

基于更新频率、数据完整性、用户评价等多维度评估数据源质量

⚙️ 快速配置

申请通过后，将以下配置添加到您的 Claude 配置文件中，即可启用可信数据源检索服务：

{ "mcpServers": { "datasource_hub": { "type": "http", "url": "${DATASOURCE_HUB_URL:-http://localhost:8001/mcp}", "headers": { "Authorization": "Bearer ${DATASOURCE_HUB_TOKEN}" } } } }

配置说明

type: HTTP 类型的 MCP 服务器
url: 数据源服务地址，支持通过环境变量配置
Authorization: 访问令牌，通过下方申请表单获取
